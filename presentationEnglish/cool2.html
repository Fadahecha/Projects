Some Acronyms for Web developers:




API Application Programming Interface: a set of pre-written code features specified by the publisher of the API. (properties, methods, classes, etc.)


CDN Content Delivery Network.


CRUD Create, Read, Update, Delete: a term used in database management, it contains the four basic functions of a persistent database.


DOM Document Object Model.


ECMAScript (ES) European Computer Manufacturers Association Script.


FOSS Free and Open-Source Software.


JSON JavaScript Object Notation.


LAMP Linux, Apache, MySQL, PHP/Python/Perl.
